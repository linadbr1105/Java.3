연산자 1.연산자와 연산식 연산 : 연산에 사용되는 표시나 기호 피연산자 : 연산 대상이 되는 데이터(리터럴) 연산식: 연산자와 피연산자를 이용하여 연산의 과정을 기술한 것 2.연산의 방향과 우선순위 연산자의 우선순위에 따라 연산된다. 동일한 우선순위의 연산자는 연산의 방향이 있다. 괄호는 최우선순위다. 3.단항 연산자 피연산자가 1개인 연산자 부호연산자, 증감연산자, 부정연산자, 비트반전 연산자가 있다. 부호 연산자: 피연산자의 부호를 결정. boolean이랑 char를 제외한 기본 타입에 사용 가능하다. 부호 연산자가 붙으면 산출값이 int 값으로 저장된다. 4.증감연산자 피연산자에 1을 더하거나 뺀다. 앞에 붙어있으면 어떤 연산보다도 먼저 수행하고 뒤에붙어있으면 어떤 연산조다 나중에 수행한다. 5.논리부정연산자 피연산자가 가지고 있는 값의 반대값을 산출한다. 6. 비트반전 연산자 byte, short, int, long 타입만 피연산자가 될 수 있다. 비트값을 반전시킨다. 틸트한 값은 int로 저장되므로 받을떄 int로 받아야한다. 이진수를 반대로 반전시켜서 값을 도출하는게 원리이다. 7.이항연산자 피연산자가 2개인 연산자로, [산술, 문자열 연걸, 대입, 비교, 논리, 비트 논리, 비트 이동]연산자가 있다. 산술 연산자: boolean 제외 모든 타입에 사용 가능. -오버플로우 : 계산한 값이 범위를 초과했을 때. (어려운 내용이니 1회독 하고 다시 볼 것!) -정확한 계산은 정수사용 : 정확하게 계산해야 할떄는 부동소수점타입(실수)을 사용하지 않는 것이 좋다. 실수가 필요하면 마지막 값 도출시에만 실수로 변환할 수 있도록 해준다. -NaN과 Infinity연산 : 1 / 0.0 = Infinity (무한) -> Infinity + 2 = Infinity 1 % 0.0 = NaN (값이 아니다) -> NaN + 2 = NaN 문자열 연결 연산자 : 피연산자 중 문자여링 있으면 문자열로 결합시킨다. 비교연산자 : 대소 또는 동등 비교를 해서 boolean타입인 true/false를 산출. 문자열 비교-String타입의 문자열은 동등 비교 연산자는 사용 가능하나, 문자열이 같은지, 다른지를 비교하는 용도는 아니다. 그 문자열이 같은 갹채로부터 온 것인지를 비교하는 것이다. 단순히 같은 문자열이라고 true는 아니다. 문자열이 같은지 비교하고 싶으면 equals()메소드를 사용해야 한다. 논리연산자 : 논리곱, 논리합, 배타적 논리합, 그리고 논리 부정 연산을 수행한다. 피연산자는 boolean 타입만 사용할 수 있다. -논리곱(&& 혹은 &) : 피연산자 둘다 true여야 연산 결과가 true -논리합 : 피연산자 둘다 false여야 연산 결과가 false -배타적논리합 : 피연산자 둘중 하나만 true일때 연산 결과가 true -논리 부정 : 피연산자의 논리값을 반대로 출력 비트연산자: 비트(bit)단위로 연산을 한다. 즉 0과 1이 피연산자가 된다. -논리곱(&) : 두 비트 모두 1일 경우에만 연산 결과가 1 -논리합(|) : 두 비트 중 하나만 1이면 연산 결과는 1 -배타적 논리합(^) : 두비트 중 하나는 1이고 다른 하나가 0일 경우 연간 결과는 1 -논리 부정(~) : 보수 비트 연산자는 피연산자를 int타입으로 자동 타입 변환 후 연산을 수행한다. (연산을 하고 난 후에는 int타입이 된다.) 비트이동연산자 : 정수데이터의 비트를 좌측 또는 우측으로 밀어서 이동시키는 연산을 수행 -a<>b : 정수 a의 각 비트를 b만큼 오른쪽으로 이동.(앞 빈공간은 최상위였던 비트로 채움) -a>>>b : 정수 a의 각 비트를 오른쪽으로 이동. (앞 빈공간은 무조건 0비트로 채움) 대입연산자 : 오른쪽 피연산자의 값을 좌측 피연산자인 변수에 저장 -단순 대입 연산자 : = 우측의 피연산자의 값을 변수에 저장 -복합 대입 연산자 : +=,-=,*=. 우측의 피연산자의 값을 변수의 값과 더한/뺀/곱한 후에 다시 변수에 저장. 삼항연산자 : 조건식에 따라 콜론 앞 뒤의 피연산자가 선택. 조건식이 true이면 앞 연산식을, 그렇지 않으면 뒤 연산식을 저장한다.
